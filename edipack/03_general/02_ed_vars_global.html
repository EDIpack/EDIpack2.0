Global Variables

Description

Contains global variables, arrays and derived types instances shared throughout the code. In particular, it contains definitions of the effective_bath and the sector data structures, loads the ed_sparse_matrix and ed_gfmatrix classes.

Quick access

Types:

sector_map, sector

Variables:

ed_file_suffix, imphloc, jh_internal, jp_internal, jx_internal, mfhloc, neigen_sector, nlevels, nnambu, ns, ns_orb, ns_ud, nsectors, sph0, sph0d, sph0dws, sph0e_eph, sph0nd, sph0ph_eph, sph0ups, uloc_internal, ust_internal

Used modules

  • ed_sparse_matrix: Types and routines for sparse matrices

  • ed_sparse_map: Routines and types for bath-impurity sparse maps

  • ed_gfmatrix: Data types for the storing of the Green's funciton

  • ed_input_vars: User-accessible input variables
